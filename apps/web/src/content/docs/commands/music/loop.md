Set the loop mode for the queue in a server.

## Usage

```sh
;loop [mode (off | song | queue)]
```

## Example

```sh
# toggle loop mode
;loop
# turn off loop mode
;loop off
# set loop mode to song
;loop song
# set loop mode to queue
;loop queue
```
