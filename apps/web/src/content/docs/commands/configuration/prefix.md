Update the prefix for a server.

## Usage

```sh
;prefix <new volume>
```

## Example

```sh
# sets the prefix to "?"
;prefix ?
```

## Required Permissions

:::note
Learn more about permissions in [Discord's FAQ](https://support.discord.com/hc/en-us/articles/206029707-Setting-Up-Permissions-FAQ).
:::
