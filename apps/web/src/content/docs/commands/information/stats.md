View a user's statistics.

## Usage

```sh
;stats [user]
```

## Examples

```sh
# view your own stats
;stats
# view @ToastedToast's stats
;stats @ToastedToast
# view 955408387905048637's stats
;stats 955408387905048637
```

## Aliases

- statistics
